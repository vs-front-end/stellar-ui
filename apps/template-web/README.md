Starter template to try [Stellar UI](https://stellar-ui-one.vercel.app) in a Vite + React app. Includes theme switching (light, dark, ocean) and example usage of components.

## Setup

From the monorepo root:

```bash
npm install
npm run dev --workspace=template-web
```

Or from this directory (after `npm install` at root):

```bash
npm run dev
```

## What's included

- **Vite + React + TypeScript**
- **@stellar-ui-kit/web** from npm, with Tailwind CSS and the Stellar preset
- **ThemeProvider** with three variants: light, dark, ocean
- Simple UI to switch themes and see components update

## Docs

Installation and configuration: [Getting Started](https://stellar-ui-one.vercel.app/docs/getting-started) in the Stellar UI docs.
